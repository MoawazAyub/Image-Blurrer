Image Blurrer

A simple project to compare the difference between adaptive mean filter and simple mean filter.

How to get started:

Load main.m. padding.m and convolve2.m should be in the same directory
running padding.m will just add a padding to the original image. If it is run multiple times multiple paddings will be added to the original image
convolve2.m will perform convolution of matrix in variable "MATRIXB"
